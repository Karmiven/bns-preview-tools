剑灵预览工具
==========================
这是一个数据读取器

项目创建于2018年9月


功能
==========================
如需获得更多信息，请查看提交日志
## 物品
获取列表   
查看指定物品 (also store/skill/map/challenge etc.) 
view item information in search interface

## 任务
获取列表
获取主线任务列表 
查看指定任务

## 文本
比较不同版本间的更新
输出文件

## 资源文件
输出文件

## 角色属性计算
此功能存在一些瑕疵   

环境
==========================
.NET 7.0 SDK    
Windows 7 更高
